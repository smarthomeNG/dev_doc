Konfiguration
=============


Im folgenden ist beschrieben, wie das Plugin **russound** konfiguriert wird. Außerdem ist im folgenden beschrieben, wie das Plugin in den Item Definitionen genutzt werden kann.


=========
Parameter
=========


Das Plugin verfügt über folgende Parameter, die in der Datei ``../etc/plugin.yaml`` konfiguriert werden:


host
====


Russound Hostname oder IP Adresse

 - Datentyp: **ip**
 - Standardwert: **127.0.0.1**


pause_item
==========


Item zum Anhalten/Fortsetzen des Plugins

 - Datentyp: **str**


port
====


Russound Port

 - Datentyp: **int**
 - Standardwert: **9621**


==============
Item Attribute
==============


Das Plugin unterstützt folgende Item Attribute, die in den Dateien im Verzeichnis  ``../items`` verwendet werden:


rus_path
========


Pfad mit Controller, Zone und System Parameter, getrennt durch "." Parameter ist * status = Zone An oder Aus: Item muss vom Typ "bool" sein * volume = Lautstärke der Zone: Item muss vom Typ "num" sein. Gültige Werte im Bereich [0..50] * bass = Bass der Zone: Item muss vom Typ "num" sein. Gültige Werte im Bereich [-10..10] * treble = Höhen der Zone: Item muss vom Typ "num" sein. Gültige Werte im Bereich [-10..10] * balance = Balance der Zone: Item muss vom Typ "num" sein. Gültige Werte im Bereich [-10..10] * turnonvolume = Lautstärke der Zone einschalten: Item muss vom Typ "num" sein. Gültige Werte im Bereich [0..50] * currentsource = Nummer der Soundquelle: Item muss vom Typ "num" sein. * mute = Ausgabeunterdrückung der zone (Mute): Item muss vom Typ "bool" sein * loudness = Loudness der zone: Item muss vom Typ "bool" sein * partymode = Party Modus Einstellung der Zone: Item muss vom Typ string sein. Gültige Werte [ON/OFF/MASTER] * donotdisturb = "Nicht Stören" Einstellung der Zone: Item muss vom Typ string

 - Datentyp: **str**

Plugin Functions
================


Das Plugin verfügt über folgende öffentliche Funktionen, die z.B. in Logiken aufgerufen werden können.


activate()
==========


Startet die Kommunikation mit Russound neu

 - Ergebnistyp der Funktion: **foo**


suspend()
=========


Schliesst vorübergehend die Kommunikation mit Russound

 - Ergebnistyp der Funktion: **foo**

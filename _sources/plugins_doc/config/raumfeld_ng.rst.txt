Beschreibung
============

Ansteuerung von Teufel Raumfeld Geräten via Raumserver und NodeRED


Anforderungen
=============




 - Minimum SmartHomeNG Version: **1.5**

Konfiguration
=============


Im folgenden ist beschrieben, wie das Plugin **raumfeld_ng** konfiguriert wird. Außerdem ist im folgenden beschrieben, wie das Plugin in den Item Definitionen genutzt werden kann.


=========
Parameter
=========


Das Plugin verfügt über folgende Parameter, die in der Datei ``../etc/plugin.yaml`` konfiguriert werden:


cycle
=====


Zeitraum in Sekunden zwischen zwei Abfragezyklen.

 - Datentyp: **int**
 - Standardwert: **60**


host
====


Raumserver Hostname oder IP Adresse

 - Datentyp: **ip**


port
====


Raumfeld Port

 - Datentyp: **int**


==============
Item Attribute
==============


Das Plugin unterstützt folgende Item Attribute, die in den Dateien im Verzeichnis  ``../items`` verwendet werden:


rf_attr
=======


Kennzeichnet die Verwendung des Items z.B. um die Lautstärke einzustellen, ein Plpaylist zu laden, ...

 - Datentyp: **str**

 - Mögliche Werte:

   - **power_state**
   - **play_state**
   - **set_mute**
   - **load_playlist**
   - **load_track**
   - **set_volume**
   - **get_mediainfo**


rf_mode
=======


Kann verwendet werden um den Wert der Lautstärkeeinstellung als relativ zu deklarieren

 - Datentyp: **str**


rf_renderer_name
================


ToDo: rf_renderer_name

 - Datentyp: **str**


rf_scope
========


Beschreibung auf welchen Bereich sich die Aktion bezieht

 - Datentyp: **str**

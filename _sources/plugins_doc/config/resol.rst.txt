Konfiguration
=============


Im folgenden ist beschrieben, wie das Plugin **resol** konfiguriert wird. Außerdem ist im folgenden beschrieben, wie das Plugin in den Item Definitionen genutzt werden kann.


=========
Parameter
=========


Das Plugin verfügt über folgende Parameter, die in der Datei ``../etc/plugin.yaml`` konfiguriert werden:


cycle
=====


Poll Cycle in Sekunden

 - Datentyp: **int**
 - Standardwert: **60**


ip
==


IP Adresse des VBUS LAN

 - Datentyp: **ipv4**


password
========


Passwort des DL2

 - Datentyp: **str**


port
====


IP port des VBUS LAN

 - Datentyp: **int**
 - Standardwert: **7053**

 - Minimalwert: **0**

==============
Item Attribute
==============


Das Plugin unterstützt folgende Item Attribute, die in den Dateien im Verzeichnis  ``../items`` verwendet werden:


resol_bituse
============


Byte Bituse Mask

 - Datentyp: **num**

 - Minimalwert: **0**

resol_command
=============


Command hex value

 - Datentyp: **str**


resol_destination
=================


Destination hex value

 - Datentyp: **str**


resol_factor
============


Ein Skalierungsfaktor pro Byte

 - Datentyp: **list**

 - Minimalwert: **0**

resol_isSigned
==============


Ein bool pro Byte, ob Vorzeichen behaftet (True) oder nicht

 - Datentyp: **list**


resol_offset
============


Byte Offset

 - Datentyp: **num**

 - Minimalwert: **0**

resol_source
============


Source hex value

 - Datentyp: **str**

Beschreibung
============

Plugin zur Ansteuerung von SmartHomeNG via Amazon Echo / Alexa


Anforderungen
=============




 - Minimum SmartHomeNG Version: **1.3**

=============

=========
Parameter
=========


Das Plugin verfügt über folgende Parameter, die in der Datei ``../etc/plugin.yaml`` konfiguriert werden:


service_host
============


IP Adresse des service Hosts

 - Datentyp: **ip**


service_port
============


Port Nummer des service Hosts

 - Datentyp: **int**
 - Standardwert: **9000**

 - Minimalwert: **0**

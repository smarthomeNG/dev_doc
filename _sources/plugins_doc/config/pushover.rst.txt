Parameter
=========


Das Plugin verfügt über folgende Parameter, die in der Datei ``../etc/plugin.yaml`` konfiguriert werden:


apiKey
======


Der API Schlüssel

 - Datentyp: **str**


device
======


Das Gerät des Benutzer

 - Datentyp: **str**


userKey
=======


Der Benutzer Schlüssel, an den die Meldungen versendet werden

 - Datentyp: **str**

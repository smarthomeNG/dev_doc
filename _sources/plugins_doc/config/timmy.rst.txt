==============
Item Attribute
==============


Das Plugin unterstützt folgende Item Attribute, die in den Dateien im Verzeichnis  ``../items`` verwendet werden:


timmy_blink_cycles
==================


Muster nach dem geblinkt wird, hier werden die Zeitabstände für die jeweiligen Schritte in Sekunden definiert, Standard: [1,1]

 - Datentyp: **list(num)**
 - Standardwert: **[1, 1]**


timmy_blink_loops
=================


Anzahl der Durchläufe bei angefordertem Blinken, 0 = Endlosschleife

 - Datentyp: **num**
 - Standardwert: **0**


timmy_blink_pattern
===================


Muster nach dem geblinkt wird, hier werden die Zielwerte definiert, Standard: [False,True]

 - Datentyp: **list**
 - Standardwert: **[False, True]**


timmy_blink_target
==================


Pfad zum gesteuerten Item

 - Datentyp: **str**


timmy_delay_off_delay_seconds
=============================


Sekunden über die Ausschaltverzögerung

 - Datentyp: **num**
 - Standardwert: **0**


timmy_delay_on_delay_seconds
============================


Sekunden über die Einschaltverzögerung

 - Datentyp: **num**
 - Standardwert: **0**


timmy_delay_target_item
=======================


Pfad zum gesteuerten Item

 - Datentyp: **str**

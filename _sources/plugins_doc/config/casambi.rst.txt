Beschreibung
============

Gateway plugin fuer Casambi


Anforderungen
=============




 - Minimum SmartHomeNG Version: **1.7**

Konfiguration
=============


Im folgenden ist beschrieben, wie das Plugin **casambi** konfiguriert wird. Außerdem ist im folgenden beschrieben, wie das Plugin in den Item Definitionen genutzt werden kann.


=========
Parameter
=========


Das Plugin verfügt über folgende Parameter, die in der Datei ``../etc/plugin.yaml`` konfiguriert werden:


api_key
=======


Casambi API key

 - Datentyp: **str**


email
=====


Emailadresse des Casambi Accounts

 - Datentyp: **str**


password
========


Passwort des Casambi Accounts

 - Datentyp: **str**


==============
Item Attribute
==============


Das Plugin unterstützt folgende Item Attribute, die in den Dateien im Verzeichnis  ``../items`` verwendet werden:


casambi_id
==========


Casambi Geraete ID

 - Datentyp: **int**

 - Minimalwert: **0**

casambi_rx_key
==============


Key fuer den Empfang von Statusinformationen

 - Datentyp: **str**

 - Mögliche Werte:

   - **ON**
   - **DIMMER**
   - **VERTICAL**
   - **CCT**
   - **BACKEND_ONLINE_STAT**


casambi_tx_key
==============


Key fuer das Senden von Kommandos ins Casambi Netzwerk

 - Datentyp: **str**

 - Mögliche Werte:

   - **ON**
   - **DIMMER**
   - **VERTICAL**
   - **CCT**

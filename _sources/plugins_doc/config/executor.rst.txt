Beschreibung
============

Ausführen von Python Statements im Kontext von SmartHomeNG v1.5 und höher


Anforderungen
=============




 - Minimum SmartHomeNG Version: **1.9**
 - Minimum Python Version: **3.8**

Konfiguration
=============


Im folgenden ist beschrieben, wie das Plugin **executor** konfiguriert wird. Außerdem ist im folgenden beschrieben, wie das Plugin in den Item Definitionen genutzt werden kann.


=========
Parameter
=========


Das Plugin verfügt über folgende Parameter, die in der Datei ``../etc/plugin.yaml`` konfiguriert werden:


script_entries
==============


Maximale Anzahl der Zeilen in der Listbox für Skripte. Bei Angabe von 0 wird die Listbox zur Dropdown Liste.

 - Datentyp: **int**
 - Standardwert: **6**

 - Minimalwert: **0**
 - Maximalwert: **50**

scripts
=======


relative Pfadangabe unterhalb *var* wo im Bedarfsfall die Skripte für das Executor Plugin abgelegt werden

 - Datentyp: **str**
 - Standardwert: **executor_scripts**

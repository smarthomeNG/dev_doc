Beschreibung
============

Ansteuerung einer LOGO SPS


Anforderungen
=============




 - Minimum SmartHomeNG Version: **1.2**

Konfiguration
=============


Im folgenden ist beschrieben, wie das Plugin **logo** konfiguriert wird. Außerdem ist im folgenden beschrieben, wie das Plugin in den Item Definitionen genutzt werden kann.


=========
Parameter
=========


Das Plugin verfügt über folgende Parameter, die in der Datei ``../etc/plugin.yaml`` konfiguriert werden:


host
====


IP-Adresse oder Hostname der LOGO

 - Datentyp: **ip**


io_wait
=======


Zeit zwischen zwei Leseanforderungen

 - Datentyp: **int**
 - Standardwert: **5**

 - Minimalwert: **1**
 - Maximalwert: **600**

port
====


Kommunikationsport der LOGO

 - Datentyp: **int**
 - Standardwert: **102**

 - Minimalwert: **0**
 - Maximalwert: **65535**

version
=======


Hardware Version der LOGO

 - Datentyp: **str**
 - Standardwert: **0BA7**

 - Mögliche Werte:

   - **0BA7**
   - **0BA8**


==============
Item Attribute
==============


Das Plugin unterstützt folgende Item Attribute, die in den Dateien im Verzeichnis  ``../items`` verwendet werden:


logo_read
=========


Lesebefehl für die LOGO SPS

 - Datentyp: **str**


logo_write
==========


Schreibbefehl für die LOGO SPS

 - Datentyp: **str**

Beschreibung
============

Item Werte bei Änderung an einen Volkszähler Server senden


Anforderungen
=============




 - Minimum SmartHomeNG Version: **1.3**

Konfiguration
=============


Im folgenden ist beschrieben, wie das Plugin **volkszaehler** konfiguriert wird. Außerdem ist im folgenden beschrieben, wie das Plugin in den Item Definitionen genutzt werden kann.


=========
Parameter
=========


Das Plugin verfügt über folgende Parameter, die in der Datei ``../etc/plugin.yaml`` konfiguriert werden:


host
====


Hostadresse (inklusive Port falls benötigt) auf der der Volkszähler HTTP Anfragen annimmt

 - Datentyp: **str**
 - Standardwert: **localhost**


url
===


URL auf der der Volkszähler die Daten vom Plugin entgegennimmt. Für {0} wird die vz_uuid des Datenpunktes substituiert

 - Datentyp: **str**
 - Standardwert: **/middleware.php/data/{0}.json**


==============
Item Attribute
==============


Das Plugin unterstützt folgende Item Attribute, die in den Dateien im Verzeichnis  ``../items`` verwendet werden:


vz_uuid
=======


Datenpunkt Identifikationsnummer vom Volkszähler Server der Daten entgegennimmt

 - Datentyp: **str**

=========
Parameter
=========


Das Plugin verfügt über folgende Parameter, die in der Datei ``../etc/plugin.yaml`` konfiguriert werden:


cycle
=====


Abfrage-Intervall für das IMAP Postfach

 - Datentyp: **int**
 - Standardwert: **300**


host
====


Adresse des IMAP Hosts

 - Datentyp: **str**


password
========


Passwort für die Anmeldung am IMAP host

 - Datentyp: **str**


port
====


Port des IMAP service (bitte 143 nutzen, fals tls deaktiviert wird)

 - Datentyp: **int**
 - Standardwert: **993**

 - Minimalwert: **0**
 - Maximalwert: **65535**

tls
===


tls zur Verschlüsselung nutzen

 - Datentyp: **bool**
 - Standardwert: **True**


trashfolder
===========


Name des Müll-Ordners auf dem IMAP-Server

 - Datentyp: **str**
 - Standardwert: **Trash**


username
========


Username für die Anmeldung am IMAP host

 - Datentyp: **str**

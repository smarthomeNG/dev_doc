Beschreibung
============

Ansteuerung von Panasonic Klimaanlagen über die Panasonic Comfort Cloud


Anforderungen
=============




 - Minimum SmartHomeNG Version: **1.10.0**

Konfiguration
=============


Im folgenden ist beschrieben, wie das Plugin **panasonic_ac** konfiguriert wird. Außerdem ist im folgenden beschrieben, wie das Plugin in den Item Definitionen genutzt werden kann.


=========
Parameter
=========


Das Plugin verfügt über folgende Parameter, die in der Datei ``../etc/plugin.yaml`` konfiguriert werden:


password
========


Login Password für die Panasonic Comfort Cloud

 - Datentyp: **str**


username
========


Login name für die Panasonic Comfort Cloud

 - Datentyp: **str**


==============
Item Attribute
==============


Das Plugin unterstützt folgende Item Attribute, die in den Dateien im Verzeichnis  ``../items`` verwendet werden:


pcc_index
=========


Index der anzusteuernden Klimaanlage

 - Datentyp: **str**


pcc_parameter
=============


Parameter der über pcc_index gewählten Klimaanlage

 - Datentyp: **str**

 - Mögliche Werte:

   - **name**
   - **temperatureInside**
   - **temperatureOutside**
   - **temperature**
   - **power**
   - **mode**
   - **fanSpeed**
   - **airSwingHorizontal**
   - **airSwingVertical**
   - **eco**
   - **nanoe**
   - **connected**

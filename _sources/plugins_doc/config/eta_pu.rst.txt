Beschreibung
============

Anbindung der REST-Schnittstelle von ETA Heizungen (http://www.eta.co.at)


Anforderungen
=============




 - Minimum SmartHomeNG Version: **1.1**

Konfiguration
=============


Im folgenden ist beschrieben, wie das Plugin **eta_pu** konfiguriert wird. Außerdem ist im folgenden beschrieben, wie das Plugin in den Item Definitionen genutzt werden kann.


=========
Parameter
=========


Das Plugin verfügt über folgende Parameter, die in der Datei ``../etc/plugin.yaml`` konfiguriert werden:


address
=======


IP Adresse der ETA Pellet Einheit

 - Datentyp: **ip**


port
====


Port der ETA Webservers

 - Datentyp: **int**
 - Standardwert: **8080**

 - Minimalwert: **0**

setname
=======


Name des Sets, welches durch dieses Plugin genutzt wird

 - Datentyp: **str**


setpath
=======


Pfad zu den vorab gesicherten Sets von CAN-Bus-uri

 - Datentyp: **str**


==============
Item Attribute
==============


Das Plugin unterstützt folgende Item Attribute, die in den Dateien im Verzeichnis  ``../items`` verwendet werden:


eta_pu_error
============


Wenn auf 'yes' gesetzt, werden Fehlermeldung der ETA Pellet Einheit in dieses Item eingelesen.

 - Datentyp: **str**

 - Mögliche Werte:

   - 
   - **yes**
   - **no**


eta_pu_type
===========


Repräsentiert das Feld der Datenzeile.

 - Datentyp: **str**

 - Mögliche Werte:

   - **strValue**
   - **unit**
   - **decPlaces**
   - **scaleFactor**
   - **advTextOffset**
   - **calc**


eta_pu_uri
==========


Enthält die CAN-Bus-Id. Die Pellet Einheit zeigt alle Ids mit Beschreibung unter der Url http://<ip>/user/menu

 - Datentyp: **str**

Beschreibung
============

Anbindung an eine UPS über den NUT daemon   Mehr über NUT unter http://networkupstools.org/


Anforderungen
=============




 - Minimum SmartHomeNG Version: **1.3**

Konfiguration
=============


Im folgenden ist beschrieben, wie das Plugin **nut** konfiguriert wird. Außerdem ist im folgenden beschrieben, wie das Plugin in den Item Definitionen genutzt werden kann.


=========
Parameter
=========


Das Plugin verfügt über folgende Parameter, die in der Datei ``../etc/plugin.yaml`` konfiguriert werden:


cycle
=====


Lesezyklus in Sekunden

 - Datentyp: **int**
 - Standardwert: **60**


host
====


Host des USV Daemons

 - Datentyp: **str**
 - Standardwert: **localhost**


port
====


IP Port des USV Daemons

 - Datentyp: **int**
 - Standardwert: **3493**

 - Minimalwert: **0**
 - Maximalwert: **65535**

timeout
=======


Telnet Lese-Timeout (Sekunden)

 - Datentyp: **int**
 - Standardwert: **5**


ups
===


Name der USV

 - Datentyp: **str**


==============
Item Attribute
==============


Das Plugin unterstützt folgende Item Attribute, die in den Dateien im Verzeichnis  ``../items`` verwendet werden:


nut_var
=======


ups Variable

 - Datentyp: **str**

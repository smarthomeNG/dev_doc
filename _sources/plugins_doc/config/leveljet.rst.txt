Beschreibung
============

Implementiert LevelJET Regenwasser Fuellstandsanzeige (www.projet.de)


Anforderungen
=============




 - Minimum SmartHomeNG Version: **1.3**

Konfiguration
=============


Im folgenden ist beschrieben, wie das Plugin **leveljet** konfiguriert wird. Außerdem ist im folgenden beschrieben, wie das Plugin in den Item Definitionen genutzt werden kann.


=========
Parameter
=========


Das Plugin verfügt über folgende Parameter, die in der Datei ``../etc/plugin.yaml`` konfiguriert werden:


baudrate
========


Baudrate (default: 19200)

 - Datentyp: **str**


serialport
==========


Pfad zu serieller USB Schnittstelle (default: /dev/ttyUSB1)

 - Datentyp: **str**


update_cycle
============


zyklische Abfrage [sek.) (default: 240)

 - Datentyp: **str**


==============
Item Attribute
==============


Das Plugin unterstützt folgende Item Attribute, die in den Dateien im Verzeichnis  ``../items`` verwendet werden:


ljet_cmd
========


Wert der von den leveljet Device in das Item eigelesen werden soll

 - Datentyp: **str**

 - Mögliche Werte:

   - **dist**
   - **level**
   - **liter**
   - **percent**
   - **outflags**

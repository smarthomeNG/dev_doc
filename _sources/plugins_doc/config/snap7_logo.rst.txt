Beschreibung
============

Ansteuerung einer Siemens LOGO PLC


Anforderungen
=============




 - Minimum SmartHomeNG Version: **1.8**

Konfiguration
=============


Im folgenden ist beschrieben, wie das Plugin **snap7_logo** konfiguriert wird. Außerdem ist im folgenden beschrieben, wie das Plugin in den Item Definitionen genutzt werden kann.


=========
Parameter
=========


Das Plugin verfügt über folgende Parameter, die in der Datei ``../etc/plugin.yaml`` konfiguriert werden:


cycle
=====


Zeit (sec) nachdem eine neue Verbindung zur Logo aufgebaut wird, um Änderungen zu holen

 - Datentyp: **num**
 - Standardwert: **5**


host
====


IP Adresse der anzusprechenden Siemens-Logo

 - Datentyp: **ipv4**


tsap_client
===========


tsap_client - Dezimalzahl

 - Datentyp: **num**
 - Standardwert: **256**


tsap_server
===========


tsap_server - Dezimalzahl

 - Datentyp: **num**
 - Standardwert: **512**


version
=======


Siemens Hardware Version 0BA7 oder 0BA8

 - Datentyp: **str**
 - Standardwert: **0BA7**

 - Mögliche Werte:

   - **0BA7**
   - **0BA8**


==============
Item Attribute
==============


Das Plugin unterstützt folgende Item Attribute, die in den Dateien im Verzeichnis  ``../items`` verwendet werden:


logo_read
=========


Eingang, Ausgang, Merker von der Siemens-Logo lesen

 - Datentyp: **str**


logo_write
==========


Eingang, Ausgang, Merker in die Siemens-Logo schreiben

 - Datentyp: **str**

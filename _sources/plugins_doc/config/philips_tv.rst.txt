Beschreibung
============

Plugin zur Anbindung eines Philips TVs an SmartHomeNG


Anforderungen
=============




 - Minimum SmartHomeNG Version: **1.8.0**

Konfiguration
=============


Im folgenden ist beschrieben, wie das Plugin **philips_tv** konfiguriert wird. Außerdem ist im folgenden beschrieben, wie das Plugin in den Item Definitionen genutzt werden kann.


=========
Parameter
=========


Das Plugin verfügt über folgende Parameter, die in der Datei ``../etc/plugin.yaml`` konfiguriert werden:


deviceID
========


Device ID (beim Bindungsvorgang automatisch generiert)

 - Datentyp: **str**


deviceKey
=========


Passwort (beim Bindungsvorgang automatisch generiert)

 - Datentyp: **str**


ip
==


IP Adresse des Philips Geraetss

 - Datentyp: **ip**


==============
Item Attribute
==============


Das Plugin unterstützt folgende Item Attribute, die in den Dateien im Verzeichnis  ``../items`` verwendet werden:


philips_tv_rx_key
=================


Key fuer den Empfang von Statusinformationen

 - Datentyp: **str**

 - Mögliche Werte:

   - **MUTE**
   - **VOLUME**
   - **ERROR**
   - **POWERSTATE**
   - **POWERSTATEINFO**
   - **CHANNEL**


philips_tv_tx_key
=================


Key fuer das Senden von Kommandos an den Philips TV

 - Datentyp: **str**

 - Mögliche Werte:

   - **MUTE**
   - **POWEROFF**

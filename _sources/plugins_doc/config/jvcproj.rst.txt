Beschreibung
============

Plugin um JVC D-ILA Projektoren zu kontrollieren und durch jvcprojectortools erstellte Gammakurven zu uebertragen


Anforderungen
=============




 - Minimum SmartHomeNG Version: **1.3**

Konfiguration
=============


Im folgenden ist beschrieben, wie das Plugin **jvcproj** konfiguriert wird. Außerdem ist im folgenden beschrieben, wie das Plugin in den Item Definitionen genutzt werden kann.


=========
Parameter
=========


Das Plugin verfügt über folgende Parameter, die in der Datei ``../etc/plugin.yaml`` konfiguriert werden:


gammaconf_dir
=============


Speicherort der Gamma- Konfigurationsdateien

 - Datentyp: **str**
 - Standardwert: **/usr/local/smarthome/etc/jvcproj/**


host
====


Die IP Adresse des JVC Projektors ist anzugeben.

 - Datentyp: **ip**
 - Standardwert: **0.0.0.0**


==============
Item Attribute
==============


Das Plugin unterstützt folgende Item Attribute, die in den Dateien im Verzeichnis  ``../items`` verwendet werden:


jvcproj_cmd
===========


Bei Aenderung des Items auf WAHR wird der angegebene Befehl (oder mehrere Befehle) an den Projektor gesendet. Die Befehle muessen durch | getrennt werden.

 - Datentyp: **str**


jvcproj_gamma
=============


Bei Aenderung des Items auf WAHR werden die Gammadaten aus der aufgefuehrten Datei in den angegebenen Custom Gammaspeicher (1-3) geladen. Die beiden Attribute muessen durch | getrennt werden.

 - Datentyp: **str**

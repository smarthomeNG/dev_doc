=========
Parameter
=========


Das Plugin verfügt über folgende Parameter, die in der Datei ``../etc/plugin.yaml`` konfiguriert werden:


auth_type
=========


Zu verwendender auth_type (0: no authentication, 1: Basic Authentication, 2: Digest Authentication)

 - Datentyp: **int**
 - Standardwert: **0**

 - Minimalwert: **0**
 - Maximalwert: **2**

intercom_ip
===========


Intercom IP Adresse

 - Datentyp: **ip**


password
========


Passwort, falls auth_type > 0

 - Datentyp: **str**
 - Standardwert: **None***


ssl
===


ssl Verschlüsselung verwenden

 - Datentyp: **bool**
 - Standardwert: **False**


username
========


Benutzername, falls auth_type > 0

 - Datentyp: **str**
 - Standardwert: **None***

Beschreibung
============

Unterstützung für Systemair residential air Geräte über Modbus.


Anforderungen
=============




 - Minimum SmartHomeNG Version: **1.3**

=============


Im folgenden ist beschrieben, wie das Plugin **systemair** konfiguriert wird. Außerdem ist im folgenden beschrieben, wie das Plugin in den Item Definitionen genutzt werden kann.


=========
Parameter
=========


Das Plugin verfügt über folgende Parameter, die in der Datei ``../etc/plugin.yaml`` konfiguriert werden:


serialport
==========


Serielle Schnittstelle (z.B. /dev/ttyUSB0)

 - Datentyp: **str**


slave_address
=============


Slave Adresse

 - Datentyp: **int**
 - Standardwert: **1**

 - Minimalwert: **1**

update_cycle
============


Update Zyklus in Sekunden

 - Datentyp: **int**
 - Standardwert: **30**

 - Minimalwert: **1**
 - Maximalwert: **600**

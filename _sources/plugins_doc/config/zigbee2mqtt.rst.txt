Beschreibung
============

Plugin zur Steuerung von Geräten, die mit einem Zigbee Gateway mit der Zigbee2MQTT Firmware versehen sind. Die Kommunikation erfolgt über das MQTT Module von SmartHomeNG.


Anforderungen
=============




 - Minimum SmartHomeNG Version: **1.8.2**
 - Minimum Python Version: **3.8**

Konfiguration
=============


Im folgenden ist beschrieben, wie das Plugin **zigbee2mqtt** konfiguriert wird. Außerdem ist im folgenden beschrieben, wie das Plugin in den Item Definitionen genutzt werden kann.


=========
Parameter
=========


Das Plugin verfügt über folgende Parameter, die in der Datei ``../etc/plugin.yaml`` konfiguriert werden:


base_topic
==========


TopicLevel_1 um mit dem ZigBee2MQTT Gateway zu kommunizieren (%topic%)

 - Datentyp: **str**
 - Standardwert: **zigbee2mqtt**


poll_period
===========


Zeitabstand in Sekunden in dem das Gateway Infos liefer soll

 - Datentyp: **int**
 - Standardwert: **300**

 - Minimalwert: **10**
 - Maximalwert: **3600**

read_at_init
============


Einlesen aller Werte beim Start

 - Datentyp: **bool**
 - Standardwert: **True**


==============
Item Attribute
==============


Das Plugin unterstützt folgende Item Attribute, die in den Dateien im Verzeichnis  ``../items`` verwendet werden:


zigbee2mqtt_attr
================


Zu lesendes/schreibendes Attribut des ZigBee2MQTT Devices. Achtung: Nicht jedes Attribut ist auf allen Device-Typen vorhanden.

 - Datentyp: **str**


zigbee2mqtt_topic
=================


TopicLevel_2 um mit dem ZigBee2MQTT Gateway zu kommunizieren; entspricht dem Friendly_Name oder Short_Name des Zigbee-Gerätes

 - Datentyp: **str**

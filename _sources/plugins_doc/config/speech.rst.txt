Konfiguration

Im folgenden ist beschrieben, wie das Plugin **speech** konfiguriert wird. Außerdem ist im folgenden beschrieben, wie das Plugin in den Item Definitionen genutzt werden kann.


=========
Parameter
=========


Das Plugin verfügt über folgende Parameter, die in der Datei ``../etc/plugin.yaml`` konfiguriert werden:


acl
===


Zugriffskontrolle

 - Datentyp: **str**
 - Standardwert: *****


config_file
===========


Konfigurationsdatei

 - Datentyp: **str**
 - Standardwert: **speech.py**


default_access
==============


Zugriffskontrolle default

 - Datentyp: **str**


listen_ip
=========


IP Adresse auf der gehört werden soll

 - Datentyp: **ip**
 - Standardwert: **0.0.0.0**


listen_port
===========


Gibt den Port an auf dem der Speech Parser auf Textpakete wartet

 - Datentyp: **int**
 - Standardwert: **2788**


==============
Item Attribute
==============


Das Plugin unterstützt folgende Item Attribute, die in den Dateien im Verzeichnis  ``../items`` verwendet werden:


sp
==


Zugriffskontrolle für den Speechparser

 - Datentyp: **str**
 - Standardwert: **ro**

Beschreibung
============

Auslesen der Web-Seite eines SolarLog


Anforderungen
=============




 - Minimum SmartHomeNG Version: **1.4**

Konfiguration
=============


Im folgenden ist beschrieben, wie das Plugin **solarlog** konfiguriert wird. Außerdem ist im folgenden beschrieben, wie das Plugin in den Item Definitionen genutzt werden kann.


=========
Parameter
=========


Das Plugin verfügt über folgende Parameter, die in der Datei ``../etc/plugin.yaml`` konfiguriert werden:


cycle
=====


Zeit zwischen zwei Updateläufen. Default ist 300 Sekunden.

 - Datentyp: **int**
 - Standardwert: **300**


fw2x
====


Firmware <= 2.x

 - Datentyp: **bool**
 - Standardwert: **True**


host
====


Hostadresse des SolarLog Gerätes

 - Datentyp: **ip**


==============
Item Attribute
==============


Das Plugin unterstützt folgende Item Attribute, die in den Dateien im Verzeichnis  ``../items`` verwendet werden:


solarlog
========


Variablendefinition für die Abfrage

 - Datentyp: **str**

Beschreibung
============

Plugin um Yamaha RX-V und RX-S Receiver via XML API zu kontrollieren


Anforderungen
=============




 - Minimum SmartHomeNG Version: **1.1**

==============
Item Attribute
==============


Das Plugin unterstützt folgende Item Attribute, die in den Dateien im Verzeichnis  ``../items`` verwendet werden:


yamaha_cmd
==========


Kommando das an den Yamaha Receiver gesendet werden soll

 - Datentyp: **str**

 - Mögliche Werte:

   - **state**
   - **power**
   - **input**
   - **volume**
   - **mute**


yamaha_host
===========


Hostname oder IP-Adresse des Yamaha Receivers mit dem kommuniziert werden soll

 - Datentyp: **ip**

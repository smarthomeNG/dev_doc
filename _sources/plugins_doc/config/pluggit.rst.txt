Beschreibung
============

Anbindung einer KWL Pluggit AP310 über das Modbus Protokoll


Anforderungen
=============




 - Minimum SmartHomeNG Version: **1.2**

Konfiguration
=============


Im folgenden ist beschrieben, wie das Plugin **pluggit** konfiguriert wird. Außerdem ist im folgenden beschrieben, wie das Plugin in den Item Definitionen genutzt werden kann.


=========
Parameter
=========


Das Plugin verfügt über folgende Parameter, die in der Datei ``../etc/plugin.yaml`` konfiguriert werden:


cycle
=====


Zykluszeit

 - Datentyp: **int**
 - Standardwert: **50**


host
====


Hostname oder IP Adresse des Pluggit Dienstes

 - Datentyp: **str**


port
====


Port Nummer des Pluggit Dienstes

 - Datentyp: **int**
 - Standardwert: **502**

 - Minimalwert: **0**
 - Maximalwert: **65535**

==============
Item Attribute
==============


Das Plugin unterstützt folgende Item Attribute, die in den Dateien im Verzeichnis  ``../items`` verwendet werden:


pluggit_listen
==============


In das Item zu lesende Daten

 - Datentyp: **str**

 - Mögliche Werte:

   - **prmRamIdxUnitMode** |_| - |_| Aktiver Modus der Einheit> 0x0004 Manueller Modus; 0x0008 Wochenprogramm
   - **prmNumOfWeekProgram** |_| - |_| Nummer des Wochenprogrammes (für Week Programmier-Modus)
   - **prmRomIdxSpeedLevel** |_| - |_| Geschwindigkeit des Lüfters im manuellen Modus; zeigt den aktuellen Gschwindigkeits-Level [4-0]
   - **prmFilterRemainingTime** |_| - |_| Verbleibende Zeit der Lebensdauer des Filters (in Tagen)
   - **prmRamIdxT1** |_| - |_| Frischluft Temperatur in °C
   - **prmRamIdxT2** |_| - |_| Zuluft Temperatur in °C
   - **prmRamIdxT3** |_| - |_| Abluft Temperatur in °C
   - **prmRamIdxT4** |_| - |_| Fortluft Temperatur in °C


pluggit_send
============


In die Pluggit Einheit zu schreibender Wert

 - Datentyp: **str**

 - Mögliche Werte:

   - **activatePowerBoost** |_| - |_| Bool Variable welche den Unit Mode auf manuellen Modus ändert und den Lüfter Gschwindikeits-Level auf den höchsten Wert (4) setzt.

Beschreibung
============

Über das CO2Meter Plugin können Daten aus dem Dostmann TFA AirCO2ntrol CO2 Messgerät via USB abgerufen werden.


Anforderungen
=============




 - Minimum SmartHomeNG Version: **1.6**

Konfiguration
=============


Im folgenden ist beschrieben, wie das Plugin **co2meter** konfiguriert wird. Außerdem ist im folgenden beschrieben, wie das Plugin in den Item Definitionen genutzt werden kann.


=========
Parameter
=========


Das Plugin verfügt über folgende Parameter, die in der Datei ``../etc/plugin.yaml`` konfiguriert werden:


device
======


Pfad zu den rohen USB Daten (optional, default: /dev/hidraw0)

 - Datentyp: **str**
 - Standardwert: **/dev/hidraw0**


time_sleep
==========


Wartezeit in Sekunden nach jeder Datenanfrage (optional, default: 5)

 - Datentyp: **int**
 - Standardwert: **5**


==============
Item Attribute
==============


Das Plugin unterstützt folgende Item Attribute, die in den Dateien im Verzeichnis  ``../items`` verwendet werden:


co2meter_data_type
==================


Der auszulesende Datentyp, "temperature" oder "co2"

 - Datentyp: **str**

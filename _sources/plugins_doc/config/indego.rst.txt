Beschreibung
============

Indego Plugin zum Auslesen von Bosch Indego Mährobotern mit Connect Ausstattung


Anforderungen
=============




 - Minimum SmartHomeNG Version: **1.4**

Konfiguration
=============


Im folgenden ist beschrieben, wie das Plugin **indego** konfiguriert wird. Außerdem ist im folgenden beschrieben, wie das Plugin in den Item Definitionen genutzt werden kann.


=========
Parameter
=========


Das Plugin verfügt über folgende Parameter, die in der Datei ``../etc/plugin.yaml`` konfiguriert werden:


img_pfad
========


Pfad und Dateiname, wo die Kartendatei gespeichert werden soll. Es müssen Schreibrechte für SmartHomeNG bestehen

 - Datentyp: **str**
 - Standardwert: **/tmp/garden.svg**


indego_url
==========


Indego Webseite für den Servicezugriff

 - Datentyp: **str**
 - Standardwert: **https://api.indego.iot.bosch-si.com/api/v1/**


parent_item
===========


Das Eltern Item für alle Kinditems, die diese Instanz betreffen

 - Datentyp: **str**
 - Standardwert: **indego**


password
========


Passwort wie in der Indego App angegeben

 - Datentyp: **str**


update_cycle
============


Zeitlicher Abstand zwischen zwei Verbindungen zur Website

 - Datentyp: **int**
 - Standardwert: **30**


user
====


Benutzername wie in der Indego App angegeben

 - Datentyp: **str**

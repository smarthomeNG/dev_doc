Beschreibung
============

Erlaubt das Senden von Notifications an Plex Clients (wie RasPlex)


Anforderungen
=============




 - Minimum SmartHomeNG Version: **1.1**

Konfiguration
=============


Im folgenden ist beschrieben, wie das Plugin **plex** konfiguriert wird. Außerdem ist im folgenden beschrieben, wie das Plugin in den Item Definitionen genutzt werden kann.


=========
Parameter
=========


Das Plugin verfügt über folgende Parameter, die in der Datei ``../etc/plugin.yaml`` konfiguriert werden:


displaytime
===========


Anzeigedauer in Millisekunden

 - Datentyp: **int**
 - Standardwert: **6000**


==============
Item Attribute
==============


Das Plugin unterstützt folgende Item Attribute, die in den Dateien im Verzeichnis  ``../items`` verwendet werden:


plex_host
=========


Hostname oder IP Adresse des Plax Hosts

 - Datentyp: **str**


plex_port
=========


Port Nummer auf dem Plex Host

 - Datentyp: **int**
 - Standardwert: **3005**

 - Minimalwert: **0**

===============
Logik Parameter
===============


Das Plugin verfügt über folgende Parameter, die in der Datei ``../etc/logic.yaml`` konfiguriert werden:

Keine Logik Parameter in den Metadaten beschrieben - **Bitte in der README nachsehen** (siehe Fußnote)

================
Plugin Functions
================


Das Plugin verfügt über folgende öffentliche Funktionen, die z.B. in Logiken aufgerufen werden können.


notify(title, message, image= 'info')
=====================================


Nachrichten an Plex senden

 - Die Funktion liefert kein Ergebnis


title
-----

Titel der Nachricht

 - Datentyp: **str**


message
-------

Nachrichten-text

 - Datentyp: **str**


image
-----

Name des Biled, welches im Nachrichten Dialgo angezeigt werden soll

 - Datentyp: **str**
 - Standardwert: **info**

 - Mögliche Werte:

   - **info**
   - **warning**
   - **error**

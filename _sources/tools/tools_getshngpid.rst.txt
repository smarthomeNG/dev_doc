getshngpid
==========

Dieses Shell Skript gibt die PID der laufenden Instanz von SmartHomeNG aus.

Diese Information wird auch auf der Seite **System** in der Admin GUI im Tab **Systemeigenschaften** angezeigt.


.. code::

   smarthome$ tools/getshngpid
   The PID of the running instance of SmartHomeNG is 23447
   smarthome$

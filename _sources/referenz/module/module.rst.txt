======
Module
======

Module erweitern die Funktionalität des Core von SmartHomeNG. Neben dem **admin** Modul implementieren sie
Interfaces, die durch Plugins genutzt werden können.

Bisher existieren die folgenden Module:

Dokumentation beschrieben.


============
Installation
============

Die hier beschriebenen Punkte erläutern die Anforderungen und beschreiben die Installation von
SmartHomeNG und benötigter oder optionaler Komponenten.

Die Komplettanleitung enthält eine Schritt-für-Schritt Anleitung für die Installation und Konfiguration
der entsprechenden Komponenten. Die Informationen zu den einzelnen Komponenten sind auch hilfreich, wenn
in einer bestehenden Installation weitere Komponenten nachinstalliert werden sollen.

Es wird auch die Installation mit Docker und das Update bzw. Upgrade von älteren Versionen beschrieben.
